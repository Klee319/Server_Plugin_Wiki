Ars Noveau

Ars Nouveau MODの魔法システムをバニラ+Geyser互換で再現するPaperプラグイン。

はじめに

ArsPaperはMinecraft MOD「Ars Nouveau」の魔法システムをPaperサーバー向けプラグインとして再実装したものです。バニラクライアントおよびGeyser（統合版）との互換性を保ちながら、本格的な魔法体験を提供します。

始め方（6ステップ）

  1. 作業台で見習いの魔法書を作成 -- 鉄の剣・鉄のツルハシ・本・鉄の斧・鉄のシャベルでクラフト
  2. 台座と刻印台を作成 -- 作業台レシピで台座（Pedestal）と刻印台（Scribing Table）をクラフト
  3. 刻印台でグリフ解放 -- 刻印台を設置して右クリック。経験値レベル＋素材を消費してグリフをアンロック
  4. スペルブックでスニーク+左クリック -- スペル作成GUIを開く
  5. 形態→効果→増強でスペルを組む -- 形態で発動方法を決め、効果を設定し、増強で強化
  6. 右クリックでスペル発動 -- 組んだスペルを使って魔法を放つ
進行の流れ
見習いの魔法書(作業台) → グリフ解放(刻印台) → スペル作成
  ↓
ソースジェム精製(儀式) → ソースストーン(作業台) → 防具・装備作成(儀式)
  ↓
魔術師の魔術書(儀式アップグレード) → 上位グリフ解放 → 大魔導士の魔導書(儀式)

スペルシステム

スペルは 形態(Form) + 効果(Effect) + 増強(Augment) の組み合わせで構成されます。

基本ルール

  • スペルの先頭は必ず 形態 1つ
  • 形態の後に 効果 を配置（複数可）
  • 各効果の後に 増強 を配置すると、直前の効果を強化
  • スペルブックのティアに応じて使えるグリフのティアが制限される
  • グリフ数の上限: 見習い3個、魔術師6個、大魔導士10個（ワンドは1スロット）

破壊(break)に減少(dampen)を付けると、ブロックを壊す代わりにユーティリティ操作を行います。範囲(aoe)増強と組み合わせれば一括操作可能。ブロック状態（階段の向き、ハーフブロックの上下等）は保持されます。

スペル作成

  1. スペルブックを手に持ち スニーク+左クリック でGUIを開く
  2. 上段にスペル構成を配置（グリフ数はスペルブックのティアで制限）
  3. 形態→効果→増強の順に左クリックで追加
  4. 右クリックで末尾から削除
  5. 歯車アイコンからスペル設定（名前変更・アイテムバインド）

Sourceネットワーク

Sourceは儀式やソースリンクの燃料となるエネルギーです。

ソースジャー

  • 最大 10,000 Source を貯蔵
  • ソースベリー を右クリックで +100 Source
  • 破壊してもSource量は保持される（再設置で復元）

ソースリンク

  • ヴォルケーニック: かまどの燃料を消費してSource生成 → 隣接するソースジャーに自動供給
  • マイセリアル: 燻製器の食料を消費してSource生成 → 隣接するソースジャーに自動供給

ドミニオンワンド

  • ソースブロック間の接続を管理（最大30ブロック距離）
  • 1つ目のブロックを右クリック → 2つ目のブロックを右クリックで接続

儀式システム

儀式コアを中心に台座を配置し、素材とSourceを消費してアイテム作成やワールドエフェクトを発動します。

儀式の手順

  1. 儀式コアを設置
  2. 周囲 3ブロック以内（上下1ブロック）に台座を配置
  3. コアと台座にレシピ通りのアイテムを設置（台座のアイテムは順不同）
  4. 近くに十分なSourceが入ったソースジャーを配置（5ブロック以内）
  5. 儀式コアを 右クリック で儀式開始（3秒のアニメーション後に完了）
