Ars Noveau

Ars Nouveau MODの魔法システムをPaperプラグインとして再実装。バニラ+Geyser互換。

はじめに

ArsPaperはMinecraft MOD「Ars Nouveau」の魔法システムをPaperサーバー向けプラグインとして再実装したものです。バニラクライアントおよびGeyser（統合版）との互換性を保ちながら、本格的な魔法体験を提供します。

始め方（6ステップ）

  1. 作業台で見習いの魔法書を作成 -- 鉄ツール5種（剣・ツルハシ・斧・シャベル・クワ）+ 本で作成
  2. 台座と刻印台を作成 -- 作業台レシピで台座（Pedestal）と刻印台（Scribing Table）をクラフト
  3. 刻印台でグリフ解放 -- 経験値と素材を消費してスペルに使うグリフをアンロック
  4. スペルブックでスニーク+左クリック -- スペル作成GUIを開く
  5. 形態→効果→増強でスペルを組む -- 形態で発動方法を決め、効果を設定し、増強で強化
  6. 右クリックでスペル発動 -- 組んだスペルを使って魔法を放つ
進行の流れ

見習いの魔法書 → グリフ解放 → ソースジェム精製 → 防具作成 → 魔術師の魔術書 → 大魔導士の魔導書

スペル作成の手順

  1. スペルブックを持ってスニーク+左クリックでスペル作成GUIを開く
  2. 最大9グリフまでスペルに組み込み可能
  3. 歯車アイコンからスペル名・アイコンなどの設定を変更
スロット切替

スニーク+右クリックで次のスロットへ、スニーク+Qで前のスロットへ切り替えます。

Sourceネットワーク

Sourceはスペル発動や儀式に必要なエネルギーです。ネットワークを構築して管理します。

ソースジャー

Sourceを貯蔵するブロックです。最大10,000 Sourceを保持できます。ソースベリーを右クリックすると+100 Sourceが補充されます。

儀式システム

手順

  1. 儀式コアを設置
  2. 周囲3ブロック以内に台座を配置
  3. レシピ通りにアイテムを台座に設置
  4. ソースジャーを近くに配置（Sourceが必要）
  5. 儀式コアを右クリックで開始（3秒後に完了）

統合版での注意点

スペル作成GUIはスニーク+左クリックで開きますが、統合版では一部操作に制限がある場合があります。アイテム表示のCustomModelDataは統合版では正しく表示されないため、disable設定を推奨します。
