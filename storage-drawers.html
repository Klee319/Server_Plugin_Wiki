Storage Drawers (PaperDrawers)

Storage Drawers 風のアイテム大量収納ブロックを Paper サーバーに追加するプラグイン。バレルブロックをベースに、1種類のアイテムを大量に格納できるドロワーを提供します。

概要

項目 詳細
対応バージョンMinecraft 1.21 / Paper
統合版対応Floodgate 連携による Bedrock Edition 互換あり
ベースブロックバレルブロック
特徴1種類のアイテムを大量に格納可能。破壊しても中身を保持したまま持ち運べる。

使い方

アイテムの格納 (右クリック)

操作 動作
右クリックアイテムを 1個 格納
Shift + 右クリック手持ちの スタック全体 を格納
ダブル右クリック (150ms以内)インベントリ内の 同種アイテムを全て 一括格納
素手で右クリックインベントリから同種アイテムを検索して 1個 格納
素手 + Shift + 右クリックインベントリから同種アイテムを検索して 1スタック 格納

アイテムの取り出し (左クリック)

操作 動作
左クリック (正面中央)アイテムを 1個 取り出し
Shift + 左クリック (正面中央)アイテムを 1スタック 取り出し
左クリック (正面の端)通常のブロック破壊
正面の外周 15% はエッジ判定となり、ブロック破壊が優先されます。

ドロワーの破壊

破壊するとシュルカーボックスのように 中身を保持したまま アイテムとしてドロップします。アイテムのツールチップに格納内容が表示されます。

ボイドドロワー

投入したアイテムを 全て消滅 させるゴミ箱ドロワーです。

  • 右クリックでアイテムを消滅（通常: 1個 / Shift: スタック全体）
  • 左クリック（取り出し）は無効
  • ホッパーから搬入されたアイテムも消滅
アイテムは復元できません。 投入前に内容をよく確認してください。

ドロワーキー

ドロワーのスロットを特定のアイテムに ロック するためのアイテムです。

操作 動作
ドロワーキーで右クリック (アイテムあり)そのアイテムタイプに ロック
ドロワーキーで右クリック (ロック済み)ロック解除
ドロワーキーで右クリック (空スロット)ロック不可
ロックされたスロットは、中身が空になってもアイテムタイプを記憶し、同じアイテムのみ受け入れます。

ホッパー連携

ドロワーはバニラのホッパーと連携します。

ホッパー → ドロワー (搬入)

ホッパーの出力先にドロワーがある場合、ホッパー内のアイテムがドロワーに自動搬入されます。

  • ドロワーのスロットが受け入れ可能なアイテムのみ搬入
  • ボイドドロワーに搬入されたアイテムは消滅

ドロワー → ホッパー (吸出し)

ドロワーの 真下 にホッパーがある場合、ドロワーからアイテムが自動的に吸い出されます。

  • バニラホッパーと同じ速度 (8tick間隔)
  • レッドストーン信号で動力を受けたホッパーは停止（バニラ準拠）
  • ボイドドロワーからは吸出し不可（中身が存在しないため）
config.yml の hopper.enabled で有効/無効を切り替えられます。

コマンド

ベースコマンド: /drawer (エイリアス: /drawers, /pd)

コマンド 説明 権限
/drawer give <player> <type> [amount] ドロワーアイテムを付与 paperdrawers.give
/drawer key [amount] ドロワーキーを自分に付与 paperdrawers.give
/drawer reload 設定をリロード paperdrawers.admin
/drawer info プラグイン情報を表示 paperdrawers.use
/drawer debug デバッグモード切り替え paperdrawers.admin

タイプ指定

<type> には以下のいずれかを指定できます。

指定方法 例
Tier番号tier1 ~ tier7
素材名basic, copper, iron, gold, diamond, netherite, creative
数字1 ~ 7
特殊void

使用例

# プレイヤーにダイヤのドロワーを1個付与
/drawer give Steve diamond

# プレイヤーにボイドドロワーを3個付与
/drawer give Steve void 3

# 自分にドロワーキーを5個付与
/drawer key 5

# 設定をリロード
/drawer reload

権限

権限ノード 説明 デフォルト
paperdrawers.use ドロワーの使用と基本コマンド 全プレイヤー
paperdrawers.give ドロワーアイテムの付与 OP
paperdrawers.admin 管理コマンド (reload, debug) OP

設定ファイル

plugins/PaperDrawers/config.yml で以下をカスタマイズ可能です。

セクション 内容
drawer-capacity各Tierの容量（スタック単位）
displayアイテム表示の有効化・距離・フォーマット
bedrock統合版互換設定
performanceキャッシュ・非同期保存
hopperホッパー連携の有効化・速度
messagesゲーム内メッセージ（日本語対応）
recipesクラフトレシピ（有効/無効・素材変更）
drawer-displayアイテム名・説明文のカスタマイズ
