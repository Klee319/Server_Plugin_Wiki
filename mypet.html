MyPet

50種類以上のモブをペットにして育成。レアリティ・性格・親密度・進化・装備・図鑑など奥深い育成要素を楽しめるプラグインです。

Paper 1.21.4+ MyPet 3.14+ LevelledMobs 4.x（任意） MythicMobs 5.x（任意）

はじめに

MyPetAddon は MyPet プラグインの拡張アドオンです。テイムしたペットにレアリティ・性格・親密度・進化・装備・図鑑などの要素を追加し、ペット育成をより奥深いものにします。

MyPet基本機能

MyPetAddon は MyPet プラグイン の上に動作します。ここではMyPet本体が提供する基本機能と、利用に必要なコマンドの使い方を説明します。

ペットの基本操作

テイムが完了すると、ペットは自動的にワールドに出現します。以降は以下のコマンドで管理します。

ペットの呼び出しと収納:

/petcall (/pc)            ペットをワールドに呼び出す（死亡後の再召喚も）
/petsendaway (/psa)       ペットをワールドから消してストレージに戻す
/petstop (/ps)            ペットに攻撃を中止させる

複数ペットの管理:

MyPetはストレージに複数のペットを保持できます。ただし同時に出せるのは1体です。

/petswitch (/psw)         ストレージ内の別のペットに切り替える（GUIが開く）
/petstore (/pst)          現在のペットをストレージに格納する
/petlist                  自分の全ペットを一覧表示

ペットの情報確認:

/petinfo (/pinfo)         ペット情報を詳しく表示（アドオン情報も付与）
/petinfo <プレイヤー名>   他プレイヤーのペット情報を表示
/petskill (/pskill)       スキル一覧と現在の値を表示

ペットの名前変更と解放:

/petname <名前>           ペットの表示名を変更する（MyPet標準機能）
/petrelease               ペットを手放す（GUI確認画面が開く。取り消し不可！）

注意: /petname はMyPet本体の「名前変更」コマンドです。テイム時の命名は /pettame を使います（後述）。

注意: /petrelease はMyPetAddonが提供するGUI版です。統合版（Bedrock Edition）でも確認画面を操作できます。

ペットのレベルと経験値

ペットは戦闘を通じて経験値（EXP）を獲得し、レベルアップします。

  • 敵を倒す: ペットが攻撃に参加した敵を倒すとEXPを獲得（飛び道具・ペット経由のキルも対象）
  • レベルアップ: 一定のEXPを獲得するとレベルが上がる
  • スキル解放: レベルが上がるとスキルツリーに応じたスキルがアンロック・強化される

レベルが高いほどペットは強力になり、進化や装備など上位コンテンツの条件にもなります。

Genius（天才）性格のペットは、スキル経験値獲得量がx1.30倍になります。

ペットの空腹

ペットには空腹度があり、定期的に餌を与える必要があります。

  • 空腹になるとペットの行動が制限されます
  • ペットの種類によって食べられるアイテムが異なります（例: オオカミ→肉類、ウシ→小麦）
  • 餌を与えると空腹ゲージが回復し、親密度EXPも獲得できます（+10 EXP）

ペットの死亡と復活

ペットがダメージを受けてHPが0になると死亡します。

  • 死亡後は一定時間経過で /petcall により再召喚可能です
  • /petrespawn で即時復活や自動復活の設定も可能です
  • 死亡時に親密度EXPが-20減少します（MyPetAddon独自ペナルティ）
  • ペットのレベル・スキル・装備は死亡しても失われません
/petrespawn (/pr)         リスポーン情報を表示
/petrespawn pay           リスポーン料金を支払って即時復活
/petrespawn auto <秒>     指定秒以上で自動支払い

行動モード

ペットにはスキルツリーのBehaviorスキルで解放される6つの行動モードがあります。

/petbehavior <モード> (/pb)    行動モードを切り替える
モード 説明 おすすめシーン
friendly一切攻撃しない街中・フレンドの近くで安全に行動
normalオーナーが攻撃した対象を攻撃デフォルトの戦闘
aggressive範囲内の全敵対Mobを自動攻撃広範囲の殲滅・レベリング
farmモンスターのみ攻撃（プレイヤー除外）通常のモブ狩り・素材集め
raidnormalと同じだがプレイヤーは攻撃しないパーティ戦闘
duel他のMyPetとのデュエル用PvPデュエル
Behaviorスキルのレベルに応じて使用可能なモードが増えます。

ペットへの騎乗

スキルツリーにRideスキルが含まれている場合、ペットに乗ることができます。

Shift + 右クリック     ペットに騎乗する
  • Controlスキルも解放されている場合、騎乗中の移動操作が可能になります
  • 上位レベルでは飛行も可能になるスキルツリーがあります（ライドLv10）

ペットのインベントリ

スキルツリーにBackpackスキルが含まれている場合、ペット用インベントリを使えます。

/petinventory (/pi)       ペットのインベントリを開く
  • 騎乗不可のペット: 右クリックでも開けます
  • 騎乗可能なペット: Shift + 右クリックで開けます
  • スキルレベルに応じて行数（最大6行）が増えます

アイテム自動拾い

スキルツリーにPickupスキルが含まれている場合、ペットが周囲のアイテムを自動回収します。

/petpickup (/pp)          ペットの自動拾いをオン/オフ切替
  • スキルレベルに応じて拾える範囲が広がります
  • 一部のスキルツリーでは経験値オーブも回収可能です

ペットのビーコン

スキルツリーにBeaconスキルが含まれている場合、ペットが周囲にバフ効果を付与します。

/petbeacon                ビーコン設定ウィンドウを開く

キャプチャーヘルパー

リードを持った状態で使用すると、モブのHPバーの下にテイム可否の表示が出ます。

/petcapturehelper (/pch)  キャプチャーヘルパーのオン/オフ切替

ペットの取引

他のプレイヤーとペットを取引できます。

/pettrade <プレイヤー名> <価格>   取引を申し込む
/pettrade accept                 取引を受け入れる
/pettrade reject                 取引を拒否する
/pettrade cancel                 取引をキャンセルする

ワールドグループ

MyPetはワールドグループ単位でペットを管理します。

  • 同じワールドグループ内ではペットを連れてワールド間を移動可能
  • 異なるワールドグループでは別のペットを使用することになります
  • ワールドグループの設定はサーバー管理者が worldgroups.yml で管理します

テイム方法

動物のテイム

動物（ウシ、ブタ、ウマ、コウモリ、アイアンゴーレム、スノウゴーレムなど）に対応するテイムアイテムを手に持ち、右クリックします。

テイムアイテム 成功率 消費
金のリンゴ5%あり
エンチャントされた金のリンゴ100%あり

テイムに成功すると、エンティティがその場で停止し（AI無効・無敵化）、レアリティ・性格が表示されます。30秒以内に /pettame <名前> コマンドで名前をつけるとペットが誕生します。

1体あたり最大5回までテイム試行できます。5回失敗するとそのモブへのテイムはロックされます。

モンスターのテイム（2段階方式）

モンスターのテイムは2つのステップが必要です。

ステップ1 -- マーキング

テイムアイテムを持ってモンスターを右クリックすると、そのモンスターがテイム対象としてマークされます。一度にマークできるのは1体のみです。

テイムアイテム 成功率 消費
ダイヤモンド5%あり
エンチャントされた金のリンゴ100%あり

ステップ2 -- トドメ＆命名

マークしたモンスターがいかなる死因であっても倒されると、その場にダミーエンティティ（動かない分身）が出現します。30秒以内に /pettame <名前> コマンドで名前をつけるとテイム完了です。

ヒント: プレイヤー自身が直接倒す必要はありません。落下ダメージ、溶岩、他のモブによるキルでもOKです。

テイムの流れ（共通）

1. テイムアイテムを持って対象モブを右クリック
   |
   v
2. アイテム消費 → 成功率判定
   |
   v
3. [動物] その場でAI停止 / [モンスター] マーキング → 倒す → ダミー出現
   |
   v
4. レアリティ・性格の表示
   |
   v
5. /pettame <名前> で命名してペット誕生！（30秒以内）

テイムできないMob

以下のMobはテイム対象外です:

  • エンダードラゴン
  • 村人
  • 行商人
ヒント: MythicMobs で生成されたカスタムモブもデフォルトでテイム除外されます。

ステータス

ステータス計算式

最終値 = ((基礎値 + 強化値) x レアリティ倍率 x 性格補正) x (1 + 親密度ボーナス%) + 装備ボーナス

計算例（ライフ）:

  • 基礎ライフ: 20、強化値: +3
  • レアリティ（レア）: x1.35
  • 性格（勇敢）: x0.95（HP-5%）
  • 親密度Lv.4のHPボーナス: +18%（加算）
  • 装備: +2
(20 + 3) x 1.35 x 0.95 = 29.5 → 29.5 + (29.5 x 0.18) + 2 = 36.8
ポイント: 親密度ボーナスはステータスごとに異なり（HP/ダメージ/速度）、レアリティ・性格適用後の値に対する割合加算です。

スキルツリー

テイム時にMyPetのスキルツリーが自動的に割り当てられます。進化時は進化先ごとに固定のスキルツリーが割り当てられます。

割り当て方式

  • テイム時: モブ種別の候補リストから重み付き抽選（モブ固有ツリーが高確率で出やすい）
  • 進化時: 進化先ごとに固定のスキルツリーが割り当てられる

抽選の優先順位:

  1. レアリティ別の設定 -- ミシック・レジェンダリーは上位ツリーが出やすい
  2. モブ種別の設定 -- そのモブの特性に合ったツリー群から抽選
  3. デフォルト設定 -- 全モブ共通のスキルツリーリストから抽選
  4. 上記すべて未設定 -- MyPet標準のランダム割り当て

すべてのモブに割り当て可能な基本スキルツリーです（9種）。

進化先にモブ固有ツリーが存在する場合、そのツリーが確定で割り当てられます。モブ固有ツリーがない進化先（ウマ系→ライド等）は汎用ツリーが固定されます。

ミシックやレジェンダリーのペットは、上位スキルツリーが出やすくなります。

スキルツリーのリロール

エコーシャードをペットに右クリックで使用すると、スキルツリーが再抽選されます。同じスキルツリーにならないよう最大2回まで再抽選を試みます。

カスタムスキルツリー（.st.jsonファイル）を plugins/MyPet/skilltrees/ に追加することも可能です。

進化

特定の条件を満たすと、ペットを別のモブタイプに進化させることができます。進化すると基礎ステータスにボーナスが付き、固定のスキルツリーが割り当てられます。

進化の手順

  1. /petevolve コマンドを実行
  2. 条件を満たす進化先がある場合、GUIが開きます
    • 進化先が1つ → 確認画面が直接表示
    • 進化先が複数 → 分岐選択画面で選んでから確認画面へ
  3. 確認画面で「進化する」を選択すると進化実行

進化条件

すべての条件を同時に満たす必要があります:

  • 最低レベル: ペットのMyPetレベル
  • 最低親密度レベル: 親密度レベル（1〜5）
  • 必要バイオーム: 特定のバイオームにいること（空欄＝制限なし）
  • 必要アイテム: インベントリ内に特定のアイテムが必要数あること

  2. プレイヤーインベントリからアイテムを装備スロットにドラッグ＆ドロップ

FAQ

動物には金のリンゴ（5%）かエンチャントされた金のリンゴ（100%）を右クリック。モンスターにはダイヤモンド（5%）かエンチャントされた金のリンゴ（100%）を右クリックしてマーキング後に倒します。どちらも最後に /pettame <名前> で名前を付けて完了です。1体あたり最大5回まで試行可能です。
ネザースターをペットに右クリックするとレアリティが1段階アップします（最大ミシックまで）。管理者は /petadmin setrarity で直接変更もできます。
アメジストの欠片をペットに右クリックすると基礎ステータスが再抽選されます。進化や強化（stat-add）で得た強化値は保持されます。
現在のバージョンでは性格の変更手段はありません。新しいペットをテイムし直す必要があります。
モブタイプが変更され、基礎ステータスにボーナスが付き、進化先に固定されたスキルツリーが割り当てられます。レベルやスキルの進行状況は保持されます。
条件を満たしていれば進化は100%成功します。条件未達の場合はGUI上に何が足りないか表示されます。
/petcall または /petrespawn pay で再召喚できます。レベル・スキル・装備は失われませんが、親密度EXPが-20減少します。
はい。MyPetはストレージに複数のペットを格納でき、/petswitch で切り替えられます。ただし同時にワールドに出せるのは1体だけです。
/petbehavior <モード> で変更できます。使用可能なモードはスキルツリーのBehaviorスキルで解放されます。デフォルトはnormalです。
スキルツリーにRideスキルが含まれていて、そのスキルが解放されている必要があります。解放後はShift+右クリックでペットに乗れます。Controlスキルも解放すると移動操作が可能になります。
スキルツリーにBackpackスキルが含まれている場合、/petinventory またはペットを右クリックで開けます。騎乗可能ペットはShift+右クリックです。
/petname はMyPet本体のコマンドで、既存ペットの表示名をいつでも変更できます。/pettame はMyPetAddonのコマンドで、テイム時に新しいペットに名前を付けて仲間にするために使います。
ペットの種類によって食べられるアイテムが異なります。例: オオカミ→肉類、ウシ→小麦。MyPetの設定で確認できます。
装備アイテムはサーバー管理者が用意するカスタムアイテムです。PDCタグ（mypetaddon:stat_*）が設定されたアイテムであれば装備としてステータスボーナスが適用されます。
